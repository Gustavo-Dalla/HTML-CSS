Tipos De Listas

Lista Ordenada

É o tipo de lista onde a ordem dos itens é importante (Tag OL), exemplo:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Se arrumar
  5. Ir para escola
  6. Almoçar...

(Tag TYPE) Escolhe o tipo de síbolo ou número que os elementos vão começar,
no caso da Lista Ordenada temos as tags A, 1, I, a, i

(Tag START) Delimita o número em que a lista começa.

Lista Não Ordenada

(Tag UL) Tipo de lista onde a ordem não importa, exemplo:

  • Feijão
  • Arroz
  • Farinha
  • Açúcar
  • Café...

(Tag TYPE) Tipos de tags para a lista não ordenada Circle, Square, Disc

Lista Mista

A lista mista é uma lista que pode-se misturar uma lista OL dentro de uma lista Ul
e vice-versa, exemplo:

  • Nintendo Wii
    • Mario
      1. Super Mario Bros
      2. Paper Mario
      3. Mario Kart
    • Wii Sports Resort
    • Need For Speed
      1. Need For Speed Underground
      2. Need For Speed Most Wanted
    • Kirby
    • Call of Duty
      1. Call of Duty Black Ops II
      2. Call of Duty Black Ops
  • PlayStation 2
    • Need For Speed
    • Black
    • Crash
      1. Crash Bandicot
      2. Crash Kart
    • Fifa
      1. Fifa 2016
      2. BOMBA PATCH
    • GTA
      1. GTA San Andreas
    • Guitar Hero
      1. Guitar Hero II

Lista de Definições

(Tag DL) A lista de definição como o nome já diz ela define um termo, como um dicionário, exemplo:

HTML (Tag DT)
Linguagem de marcação para o desenvolvimento de conteúdo de um site (Tag DD)
CSS (Tag DT)
Linguagem de marcação para o desenvolvimento do design de um site (Tag DD).
JavaScript (Tag DT)
Linguagem de programação para a criação de interatividade em um site (Tag DD).

Para lembrar melhor:
DL: Definition List
DT: Definition Term
DD: Definition Description
