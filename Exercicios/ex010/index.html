Links Externos

A tag A faz com que a frase ou a palavra envelopada leve a outro site, exemplo:
Meu canal do youtube.
Com a tag TARGET="_blank" REL="rel=external", ao clicar no hyperlink ela desvia para outra pagina em branco.

Links Internos

Pode-se também criar um HyperLink a um arquivo interno, ou seja arquivo que está em nosso computador, exemplo:
Segunda página do meu site.

Links Internos - Pasta

Pode-se também criar uma página dentro de uma pasta, para usar no HyperLink, exemplo:
Terceira página do meu site.

Links Para Download

Para criar um HyperLink para download dentro da tag A usa-se as tags DOWNLOAD = "*endereço do arquivo*" TYPE = "*media type*" (Material cap 10 tem os media types), exemplo:

  • Teste de Download-Word
  • Teste de Download-Word.zip
