Frase Normal

        Frase
           Escrita
Com a tag            PRE

Normalmente usa-se as tags CODE e PRE para códigos, exemplo portugol: 

        
int num1, num2, resultado
    escreva("Informe um numero: ")
    leia(num1)
    escreva("Informe outro numero: ")
    leia(num2)
    resultado: num1+num2
    se(resultado > 3){
        escreva("Numero maior que três")
    }
    senao{
        escreva("Numero menor que três")
    }

Citações Simples

Para fazer citações usa-se a tag Q, ou seja as aspas,
exemplo: 
A tag Q é importante pois ela é uma tag SEMÂNTICA

Citações Compostas

Para citações maiores usa-se a tag BLOCKQUOTE como um pedaço de um livro. Exemplo:

Pedaço de um livro que eu estou inventando que nem existe kk
Já a tag CITE entra dentro da tag blockquote para colocar o link de referência.

Abreviação

Para palavras abreviadas usa-se a tag ABBR que ela quando passa o cursor em cima aparece o significado
da palavra, exemplo: CSS

Tag Inutil (Ou Não)

usa-se a tag BDO para colocar uma palavra ou frase ao contrário, exemplo: Bom dia o sol já nasceu lá ná fazendinha
